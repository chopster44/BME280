# BME280
An arduino i2c driver for the Bosch BME280 humidity sensor.

The only dependencies of this driver is `Wire.h` and `Arduino.h`.
## Instalation

To install add:
`https://github.com/chopster44/BME280.git` to your `platformio.ini` file.
